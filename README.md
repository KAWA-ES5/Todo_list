# TODO管理アプリケーション

## 1. 概要
タスク管理アプリ

## 2. 実装済み機能
- タスクの追加（タイトル・詳細）
- タスク一覧表示とフィルタリング（すべて・完了・未完了）
- タスクの編集（タイトル・詳細）
- タスクの完了/未完了の切り替え（タスク左のチェックボタン）
- タスクの削除（確認画面有）
- インポート/エクスポート機能(JSONファイル)

## 3. セットアップ手順
### 必要なもの
- Python 3.x
- pip

### インストールと起動
1. ライブラリのインストール:
   pip install -r requirements.txt

2. サーバーの起動:
   uvicorn main:app --reload
